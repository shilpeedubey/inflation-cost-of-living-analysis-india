# Inflation and Cost of Living Analysis in India (2015–2024)

## Overview
This project analyzes inflation and cost-of-living trends in India using Consumer Price Index (CPI) data from MoSPI for the period 2015–2024.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Dashboard Development
- Data Visualization

## Key Insights
- Analyzed 120 monthly CPI observations.
- Compared Overall, Food, and Fuel inflation trends.
- Identified 2022 as the highest inflation year at 7.4%.
- Compared Pre-COVID, COVID, and Post-COVID inflation periods.

## Dashboard Features
- KPI Cards
- Trend Analysis
- Interactive Excel Dashboard
